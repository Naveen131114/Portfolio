# Portfolio

A premium React + Vite portfolio starter with animated sections, dark gradient visuals, and card-based project layout.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open the local Vite URL shown in the terminal.

## Included Features

- React + Vite setup
- Framer Motion animations
- Hero, About, Skills, Projects, Contact sections
- GitHub project links
- Resume download button
- Public project screenshots in `/public/projects/`
